# snapyt! - YouTube Downloader Electron App

![snapyt! Demo](https://github.com/saadnkhawaja/snapyt---youtube-downloader-electronjs/blob/main/snapyt-demo.gif)

An Electron.js desktop application that allows users to download YouTube videos seamlessly using the powerful `yt-dlp` backend. It provides a simple and user-friendly interface for selecting video quality, format, and managing downloads efficiently.

## Features

- **Smart Quality Selection:** Download videos using the exact quality options available on YouTube.
- **Multiple Format Support:** Save content as video or audio using formats like MP4, MKV, WebM, and audio-only output.
- **Fast yt-dlp Backend:** Reliable downloads powered by `yt-dlp`.
- **Queue and Download Management:** Add items to queue, monitor progress, and manage recent downloads easily.
- **Built-In Library:** Search, filter, sort, and manage downloaded media from within the app.
- **Clipboard Workflow:** Automatically detect copied YouTube links and fetch them instantly for quicker downloads.

## How to Use

1. Clone or download the repository.
2. Install dependencies with `npm install`.
3. Run the app using `npm run dev` for development or `npm start` for normal use.
4. Paste a YouTube link into the input field and click `Snapyt!`.
5. Select the available quality and format, then start your download.
